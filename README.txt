LetakMaster v1.02 Release
===========================

To run the application:
1. Double-click "start_servers.bat".
2. Select Option [1] to Start Servers.
3. The Backend API and Frontend GUI windows will launch.
4. Access the application at http://localhost:5173

Prerequisites:
- Windows 10/11
- Adobe Photoshop (for automation features)
- Microsoft Excel (for workspace features)

Note: All dependencies are included in this folder. No installation required.
